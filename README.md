# NES Brainfuck

![Brainfuck](/info/title.png)

Brainfuck VM/"compiler" for NES. Uses ca65 macroses to generate binary for supplied Brainfuck code. Read operator is not supported.

Download NES file [here](game.nes).

## Examples

```
; Hello World
BF_run:
BF_compile "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++."
jsr BF_run
```
Result:
